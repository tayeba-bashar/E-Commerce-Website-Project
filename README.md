**Public User**

**Singed In User**

- Add to cart
- Search product by category or brand
- Create order
- Procced to Payment
- Get Order Details

**Admin**
- Add New Product
- Add New Brand
- Add New Category
- Update existing product/category/brand
- Deelete existing product/category/brand
- See customer details
- Monitor order details & confirmation
